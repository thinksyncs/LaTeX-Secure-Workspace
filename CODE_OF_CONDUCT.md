# Code Of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

- Be respectful and constructive.
- Assume good intent and focus on actionable feedback.
- Report security concerns privately using the process in [`SECURITY.md`](./SECURITY.md).

## Scope

This Code of Conduct applies to project spaces, issue discussions, pull requests, release discussions, and other public interactions around this repository.

## Reporting

Please report Code of Conduct concerns to `develop@toppymicros.com`.

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that are not aligned with this Code of Conduct.
