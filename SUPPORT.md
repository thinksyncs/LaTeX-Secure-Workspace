# Support

## Security Reports

For vulnerability reports, use the process in [`SECURITY.md`](./SECURITY.md).

## Questions And General Support

- Use GitHub issues for reproducible bugs and feature requests.
- Include your VS Code version, extension version, operating system, and a minimal reproduction when possible.

## What To Include In Bug Reports

- The `.tex` root file layout you used
- The build command or command palette action you ran
- Relevant log output from the extension
- Whether the issue reproduces with a fresh VS Code window
